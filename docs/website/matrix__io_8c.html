Detailed Description

Input/output operations for matrices.

This file contains functions for input/output manipulation of matrices. Specifically, it allows populating a matrix with data from a file, printing a matrix, etc.

int CreateMatrixFromFile ( const char * filename,
Matrix ** matrix )

Creates a matrix from data in a file.

Parameters
filename- The name of the file.
matrix- Matrix that will contain the data of the matrix.
Return values
`CANNOT_OPEN_FILE`- Failure to open the file.
`INVALID_MATRIX_OR_INDICES`- The matrix or the provided indices are invalid.
`MEMORY_ALLOCATION_FAILURE`- Failure to allocate memory for the new matrix element.
`FILE_READ_ERROR`- Error reading data from the file.
`SUCCESS`- Operation successful.
